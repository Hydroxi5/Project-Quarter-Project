The 1st Dimension: Time

The Aspect

Time is among the most dangerous of elements, second only to Void in destructive potential. In a state of slowed or stopped time, it is possible to accelerate objects to beyond physically possible speeds. Nuclear fusion is a possibility for a moving object once released from the timeflow. Dispite the name of the aspect, users cannot travel backwards in time. Thankfully, this is its own safety measure, preventing possible future changes and paradoxes. Rarely, an expert Time wielder catches glimpses of the future. It is theorized that with more power, a Time wielder could see all possible futures.
